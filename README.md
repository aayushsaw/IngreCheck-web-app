## üöÄ Overview

**IngreCheck** is a modern web application designed to help consumers analyze food and cosmetic products instantly. By simply scanning a barcode, users receive a comprehensive breakdown of ingredients, nutritional quality (Nutri-Score), and environmental impact (Eco-Score).

Built with a **"Biological Tech"** aesthetic, the app features a dark, immersive interface with glassmorphism effects, ensuring a premium user experience on both desktop and mobile devices.

## ‚ú® Key Features

-   **üîç AI-Powered Scanning**: Advanced barcode and QR code detection using `html5-qrcode`.
-   **üìä Smart Analysis**: Instant decoding of additives, allergens, and nutritional values.
-   **üì± Mobile-First Design**: Fully responsive UI with a high-tech HUD overlay for scanning.
-   **‚ö° Real-time Feedback**: Live detection indicators and laser scanning animations.
-   **üé® Premium UI/UX**: Dark mode, smooth animations, and glassmorphic components.
-   **üåç Eco-Conscious**: Visibility into the environmental footprint of products.

## üöÄ Getting Started

To run this project locally, follow these steps:

### Prerequisites

-   Node.js 18+ installed
-   npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/aayushsaw/IngreCheck-web-app.git
    cd IngreCheck-web-app
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Open the app**
    Visit [http://localhost:3000](http://localhost:3000) to view the application.
