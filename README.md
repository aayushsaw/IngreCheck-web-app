# IngreCheck

**Know What You Consume.**

IngreCheck is a modern, AI-powered web application designed to help users analyze food and cosmetic products. By scanning barcodes, users get instant insights into ingredients, nutritional quality, and health impact, presented in a premium "Biological Tech" interface.
